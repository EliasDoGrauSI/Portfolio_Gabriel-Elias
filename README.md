- ⚡ Build otimizado com Vite
- 📱 Mobile-friendly
- 🎨 Paleta de cores profissional

## 🚀 Como Usar

### Pré-requisitos
- Node.js (v14 ou superior)
- npm ou yarn

### Instalação

1. Clone ou acesse o repositório:
```bash
cd Portfolio_Gabriel-Elias
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Abra seu navegador e acesse:
```
http://localhost:3000
```

### Build para Produção

Para criar uma versão otimizada para produção:
```bash
npm run build
```

Para visualizar o build localmente:
```bash
npm run preview
```

### Cores
As cores principais podem ser ajustadas no arquivo `src/styles/index.css`:

```css
  --secondary-color: #0f172a;
  --accent-color: #06b6d4;
  --text-light: #f1f5f9;
  --text-dark: #0f172a;
}
```
